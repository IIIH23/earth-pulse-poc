# Security

Do not report security vulnerabilities through public issues.

Use GitHub's private vulnerability reporting for this repository. If that
feature is unavailable, contact the repository owner privately.

Never commit:

- API keys or access tokens;
- mobile signing keys or provisioning profiles;
- `.env` files;
- production service credentials;
- private seismic-provider credentials;
- user analytics or personally identifiable information.

The public Earth Pulse feed contains only detector output and must not contain
user data.
